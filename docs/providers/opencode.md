# OpenCode Zen

**Setup complexity:** Simple
**Auth choice:** `opencode-zen`
**CLI flag:** `--opencode-zen-api-key`

## Getting an API Key

1. Go to the OpenCode platform
2. Create an account or sign in
3. Navigate to API management
4. Generate an API key and copy it

## Configuration in PapayaClaw

Select **OpenCode Zen** in the deploy wizard and paste your API key. Enter any model available on the OpenCode Zen catalog using the `opencode/` prefix format.

## Available Models

Pass-through provider. Use any model ID available on the OpenCode Zen catalog with the `opencode/` prefix.

## Notes

- Custom provider with its own model catalog.
