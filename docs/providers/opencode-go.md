# OpenCode Go

**Setup complexity:** Simple
**Auth choice:** `opencode-go`
**CLI flag:** `--opencode-go-api-key`

## Getting an API Key

1. Go to the OpenCode platform
2. Create an account or sign in
3. Navigate to the Go catalog section
4. Generate an API key and copy it

## Configuration in PapayaClaw

Select **OpenCode Go** in the deploy wizard and paste your API key. Enter any model available on the OpenCode Go catalog using the `opencode-go/` prefix format.

## Available Models

Pass-through provider. Use any model ID available on the OpenCode Go catalog with the `opencode-go/` prefix.

## Notes

- Separate catalog from OpenCode Zen with its own set of models.
