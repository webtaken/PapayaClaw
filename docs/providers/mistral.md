# Mistral

**Setup complexity:** Simple
**Auth choice:** `mistral-api-key`
**CLI flag:** `--mistral-api-key`

## Getting an API Key

1. Go to [console.mistral.ai](https://console.mistral.ai/)
2. Create an account or sign in
3. Navigate to **API Keys**
4. Create a new key and copy it

## Configuration in PapayaClaw

Select **Mistral** in the deploy wizard and paste your API key.

## Available Models

| Model ID | Display Name |
|----------|-------------|
| `mistral-large-latest` | Mistral Large |

## Notes

- Mistral is a European AI company based in Paris.
- Mistral Large is their most capable model, suitable for complex reasoning and code generation.
