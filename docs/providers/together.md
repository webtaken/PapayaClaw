# Together AI

**Setup complexity:** Simple
**Auth choice:** `together-api-key`
**CLI flag:** `--together-api-key`

## Getting an API Key

1. Go to [api.together.xyz](https://api.together.xyz/)
2. Create an account or sign in
3. Navigate to **API Keys** in Settings
4. Copy your API key

## Configuration in PapayaClaw

Select **Together AI** in the deploy wizard and paste your API key.

## Available Models

| Model ID | Display Name |
|----------|-------------|
| `together/meta-llama/Llama-4-Maverick-17Bx128E-Instruct-FP8` | Llama 4 Maverick (FP8) |

Together AI hosts many open-source models. Use the `together/` prefix followed by the model path. Browse models at [together.ai/models](https://www.together.ai/models).

## Notes

- Together AI specializes in hosting open-source models with fast inference.
- Competitive pricing for open-source model inference.
- Supports models from Meta (Llama), Mistral, and many others.
