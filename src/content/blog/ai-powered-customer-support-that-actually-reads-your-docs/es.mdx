## Ejemplo Concreto: Integración de Documentación SaaS

Digamos que tienes documentación almacenada en una base de datos de Notion. Así es como la conectas:

```javascript
import { NotionAPI } from 'notionapi';

const notion = new NotionAPI({
  authToken: process.env.NOTION_TOKEN
});

async function buscarDocumentacion(consulta) {
  // Buscar en todos los docs
  const resultados = await notion.search({
    query: consulta,
    filter: {
      property: 'Tipo',
      select: { equals: 'Documentación' }
    }
  });
  
  // Extraer páginas relevantes
  const paginas = [];
  for (const resultado of resultados.results) {
    const pagina = await notion.getPage(resultado.id);
    paginas.push({
      title: pagina.properties.title.title[0].plain_text,
      content: pagina.content,
      url: pagina.url
    });
  }
  
  return paginas;
}
```

Cuando un usuario pregunta "¿Cómo configuro autenticación de dos factores?", tu agente:

1. Llama a buscarDocumentacion("autenticación de dos factores")
2. Obtiene páginas relevantes de tus documentos
3. Responde usando solo esas fuentes
4. Cita la página de documentación específica
