# Welcome to PapayaClaw

Welcome to **PapayaClaw** blog! 🚀

What is papayaclaw? It is a platform where you can deploy your own OpenClaw agent in just three simple steps:

1. Choose your preferred communication channel: **Telegram**, **Discord**, or **WhatsApp** (currently we only support Telegram, but we'll continue adding more communication channels).
2. Set your model API key and the model you want to use (**Claude Sonnet 4.6**, **GPT-5.2**, **MiniMax 2.5**, etc).
3. Set your Telegram Bot API Key and clic deploy 🚀.
